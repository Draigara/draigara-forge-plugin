# Contract schemas

`schemas/mcp/v1` contains pinned copies of the CLI-owned Forge MCP schemas used only for consumer contract tests. Their source release and digest must be recorded when copied. Changes originate in `draigara-forge-cli`; this repository must not evolve a competing contract.

`forge.schema.v1.json` remains a consumer copy of the CLI-owned repository configuration schema and follows the same pinning rule.
